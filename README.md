💈 Barbershop-BE (Work in Progress)
==========

Backend REST API for managing barbershop appointments, built with Java Spring Boot.

The system handles authentication, appointment scheduling, feedback management, and role-based access for clients and barbers.

This backend powers the frontend application by providing secure REST endpoints, database persistence, and business logic.

The project demonstrates REST API design, layered architecture, authentication, and database integration using modern Java backend technologies.

🚀 Features
===========

### 🔐 Authentication

User login system

Role-based access (Client / Barber)

Secure route protection

### 📅 Appointment Management

Schedule new appointments

View upcoming appointments (coming soon)

View past appointment history (coming soon)

Manage personal bookings (coming soon)

* Create and manage appointments (manage coming soon)
